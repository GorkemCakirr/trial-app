# Getting Started

Getting started with Bucket UI is simple and requires no account creation. Here's what you need to do:

1. **Download the Application**

   - Download our desktop application [here](https://)
   - Start with a 7-day free trial of our Pro plan

2. **Set Up AWS CLI**
   - Install the AWS CLI on your system
   - Log in to your AWS account using SSO credentials in your terminal
   - Bucket UI will automatically detect your AWS profiles

> **Important:** You must be logged in to AWS with SSO credentials to use Bucket UI. The application will not function without proper AWS authentication.
