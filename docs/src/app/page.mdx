# Bucket UI Documentation

Welcome to Bucket UI - your intuitive desktop application for managing AWS S3 storage with ease!

## What is Bucket UI?

Bucket UI is a powerful desktop application designed to simplify your AWS S3 storage management. With Bucket UI, you can:

- Upload and download files
- Preview files directly in the application
- Edit and save files in GZIP format
- Manage your S3 buckets efficiently
