# Navigation

- The application implements a Mac-like folder structure for bucket contents.
- Users can view up to 3 levels of the bucket hierarchy simultaneously.
- Navigation is intuitive and follows standard file system conventions.
- User can scroll in if there are more than 3 folder structure inside a bucket.
